## 🧠 16.1 What is a Feature Map?

When an image passes through a convolution layer, the output is called a **feature map**.

👉 Simple meaning:

> A feature map shows **what the CNN has detected** in the image

---

### 🧠 Real-life analogy

Imagine using different glasses:

* One highlights edges
* One highlights colors
* One highlights shapes

👉 Each “view” = one feature map

## 🔍 16.8 Interpreting Feature Maps

👉 Bright areas:

* Strong feature detected

👉 Dark areas:

* No feature

---

### 🧠 Example

If detecting edges:

* Edge areas → bright
* Flat areas → dark

## 🧪 16.9 Visualizing Feature Maps (PyTorch)

```python
import torch
import matplotlib.pyplot as plt

# Get one image
image, _ = train_loader.dataset[0]
image = image.unsqueeze(0)

# Pass through first conv layer
feature_maps = model.conv1(image)

# Plot first 4 feature maps
for i in range(4):
    plt.subplot(1, 4, i+1)
    plt.imshow(feature_maps[0, i].detach(), cmap='gray')
    plt.axis('off')

plt.show()
```

---

👉 This helps students:

* See what CNN is “looking at”

## 🧠 16.11 Deep Insight

CNN does NOT understand images like humans.

👉 It understands:

* patterns
* intensities
* structures

---

## 🔍 16.12 Why Multiple Feature Maps Matter

One filter is not enough.

👉 Multiple filters allow:

* richer understanding
* better accuracy

## 🎯 16.13 Key Takeaways

* Feature map = output of convolution
* Each map detects a specific pattern
* Deeper layers learn more complex features

---

👉 Most important idea:

> CNN learns by building layers of feature maps
