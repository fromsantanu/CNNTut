## 🧠 8.1 Why Do We Need a Dataset?

Till now, we learned how CNN works.
But a model cannot learn without **data**.

👉 Data = examples from which the model learns patterns

---

### 🧠 Real-life analogy

Like a student:

* Learns math by solving many problems
* Learns images by seeing many examples

👉 Same with CNN:

> More examples → better learning

* Handwritten digits from **0 to 9**

---

## 📊 8.3 Dataset Details

| Property     | Value     |
| ------------ | --------- |
| Total images | 70,000    |
| Training set | 60,000    |
| Test set     | 10,000    |
| Image size   | 28 × 28   |
| Type         | Grayscale |

---

👉 Each image:

* Is small (28×28 pixels)
* Contains one digit

---

## 🧠 8.4 What the Model Learns

The CNN will learn:

* Shapes of digits
* Curves and edges
* Differences between numbers

---

👉 Example:

* “1” → straight line
* “0” → circular shape
* “8” → two loops

---

## 📐 8.5 Structure of MNIST Data

Each image:

```
Shape = (28, 28)
```

👉 After adding channel (for CNN):

```
(1, 28, 28)
```

👉 Batch format:

```
(batch_size, channels, height, width)
```

---

## 🎯 8.6 Input and Output

### Input:

* Image (28×28)

### Output:

* Label (digit 0–9)

---

👉 Example:

```
Image → "5"
```

### 🔹 Training Data

---

### 🔹 Test Data

* Used to **check performance**

---

👉 Important:

> Model should perform well on unseen data

## 🧪 8.8 Loading MNIST in PyTorch

```python
import torch
from torchvision import datasets, transforms

# Transform: convert image to tensor
transform = transforms.ToTensor()

# Load training data
train_data = datasets.MNIST(
    root='./data',
    train=True,
    download=True,
    transform=transform
)

# Load test data
test_data = datasets.MNIST(
    root='./data',
    train=False,
    download=True,
    transform=transform
)

print("Training samples:", len(train_data))
print("Test samples:", len(test_data))
```

## ⚙️ 8.10 Creating DataLoader

```python
from torch.utils.data import DataLoader

train_loader = DataLoader(train_data, batch_size=32, shuffle=True)
test_loader = DataLoader(test_data, batch_size=32)
```

---

👉 Why DataLoader?

* Loads data in batches
* Makes training faster
