## 🔄 9.2 CNN Pipeline (Quick Recall)

Before coding, remember:

```text
Image → Conv → ReLU → Pool → Flatten → Fully Connected → Output
```

👉 We will now build exactly this.

---

## ⚙️ 9.3 Import Required Libraries

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
```

---

## 🧱 9.4 Step 1: Define the CNN Model

In PyTorch, we create a class.

```python
class CNNModel(nn.Module):
    def __init__(self):
        super(CNNModel, self).__init__()
```

---

## 🔍 9.5 Step 2: Add Layers

---

### 🔹 Convolution Layer 1

```python
self.conv1 = nn.Conv2d(
    in_channels=1,   # MNIST = grayscale
    out_channels=8,  # number of filters
    kernel_size=3
)
```

👉 Output:

* Learns basic features (edges)

---

### 🔹 Pooling Layer

```python
self.pool = nn.MaxPool2d(2)
```

👉 Reduces size

---

### 🔹 Convolution Layer 2

```python
self.conv2 = nn.Conv2d(8, 16, 3)
```

👉 Learns deeper features

---

### 🔹 Fully Connected Layers

```python
self.fc1 = nn.Linear(16 * 5 * 5, 64)
self.fc2 = nn.Linear(64, 10)
```

👉 Final output = 10 digits (0–9)

---

## 🧠 9.6 Step 3: Forward Pass

This defines how data flows.

```python
def forward(self, x):
    x = self.pool(F.relu(self.conv1(x)))
    x = self.pool(F.relu(self.conv2(x)))
    
    x = x.view(-1, 16 * 5 * 5)  # Flatten
    
    x = F.relu(self.fc1(x))
    x = self.fc2(x)
    
    return x
```

---

## 🧪 9.7 Create the Model

```python
model = CNNModel()
print(model)
```

---

## 📐 9.8 Understanding Output Sizes (Important)

Let’s track size step-by-step:

| Step    | Output Size  |
| ------- | ------------ |
| Input   | 1 × 28 × 28  |
| Conv1   | 8 × 26 × 26  |
| Pool    | 8 × 13 × 13  |
| Conv2   | 16 × 11 × 11 |
| Pool    | 16 × 5 × 5   |
| Flatten | 400          |
| FC      | 10           |

---

👉 This is why we used:

```python
16 * 5 * 5
```

---

## 🧠 9.9 Real-Life Understanding

Think of the model like:

* First layer → sees simple patterns
* Second layer → sees complex shapes
* Final layer → decides digit

## ⚠️ 9.11 Common Mistakes

### ❌ Wrong flatten size

* Causes error
* Always calculate carefully

---

### ❌ Missing ReLU

* Model becomes weak

---

### ❌ Wrong input shape

* CNN expects:

```python
(batch, channels, height, width)
```
