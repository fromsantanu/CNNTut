## ⚙️ 19.4 Step 1: Data Preprocessing

```python
from torchvision import datasets, transforms
from torch.utils.data import DataLoader

transform = transforms.Compose([
    transforms.Resize((224, 224)),
    transforms.ToTensor()
])

train_data = datasets.ImageFolder('data/train', transform=transform)
test_data = datasets.ImageFolder('data/test', transform=transform)

train_loader = DataLoader(train_data, batch_size=32, shuffle=True)
test_loader = DataLoader(test_data, batch_size=32)
```

---

👉 Why resize to 224×224?
Because pretrained models expect this size.

---

## 🧠 19.5 Step 2: Use Transfer Learning

Instead of building from scratch, we use:
👉 ResNet

---

### 🔧 Load Pretrained Model

```python
import torchvision.models as models
import torch.nn as nn

model = models.resnet18(pretrained=True)
```

---

### 🔒 Freeze Layers

```python
for param in model.parameters():
    param.requires_grad = False
```

---

### 🔄 Replace Final Layer

```python
model.fc = nn.Linear(model.fc.in_features, 2)
```

👉 Output:

* 2 classes (Normal, Pneumonia)

---

## 🧠 19.6 Step 3: Define Loss and Optimizer

```python
import torch.optim as optim

criterion = nn.CrossEntropyLoss()
optimizer = optim.Adam(model.fc.parameters(), lr=0.001)
```

---

## 🔁 19.7 Step 4: Training Loop

```python
epochs = 5

for epoch in range(epochs):
    for images, labels in train_loader:
        
        outputs = model(images)
        loss = criterion(outputs, labels)
        
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
    
    print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")
```

---

## 📊 19.8 Step 5: Model Evaluation

```python
correct = 0
total = 0

with torch.no_grad():
    for images, labels in test_loader:
        outputs = model(images)
        _, predicted = torch.max(outputs, 1)
        
        total += labels.size(0)
        correct += (predicted == labels).sum().item()

accuracy = 100 * correct / total
print(f"Test Accuracy: {accuracy:.2f}%")
```
